Tag Input

A tag input is an input box that automatically creates tags – also called tokens – out of typed text every time a certain key is pressed. It's useful for tagging and highlighting information right on the input box.

Properties:
data-items: array of items for autosuggest.
data-field: defines property in object by which to filter items.
data-key-property: defines property used as the id of the items.
data-freetext: allows free input of items that are not in suggestion list.
data-on-change: callback for when the tags are added/removed
data-replace-spaces-with-dashes: Flag indicating that spaces will be replaced with dashes (defaults to true).
data-min-length: sets the minimum allowed length of a tag to be valid (defaults to 3).
data-max-length: sets the maximum allowed length of a tag to be valid (defaults to 9007199254740991).
data-placeholder: Placeholder text for the control.
data-max-results-to-show: Maximum number of results to be displayed at a time. If < 1 then show all results.

If you don't define data-items button will not be visible.
