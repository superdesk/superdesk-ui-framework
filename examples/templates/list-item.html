List item group

Structure for list elements groups. The content inside the elements serves just as an example.

Default group (no space between list items): add wrapper div with class sd-list-item-group.

If you need some space between list items, add modifier to default one, so the class is sd-list-item-group sd-list-item-group--space-between-items.
